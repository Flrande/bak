# Documentation

This repository uses an agent-first documentation model.

## 30-Second Start

If you are an end user, paste this quickstart URL to your coding agent:

- `https://raw.githubusercontent.com/Flrande/bak/refs/heads/master/docs/user/quickstart.md`

The quickstart is the only setup source of truth.

## Source Of Truth

- RPC method and type definitions: `packages/protocol/src/types.ts`
- JSON schema: `packages/protocol/schemas/protocol.schema.json`
- CLI command surface: `packages/cli/src/bin.ts`
- Daemon bootstrap: `packages/cli/src/server.ts`

## Redundancy Rules

- Quickstart owns first-time setup instructions.
- `cli-guide.md` owns command catalog and examples.
- `agent-prompts.md` owns prompt templates only.
- Other docs link to these sections instead of duplicating setup flows.
